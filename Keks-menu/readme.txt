Installtion guide: 

Choose a version you wanna use (latest version is recommended (0.4.7.4)(you can also use the latest beta but it could be unstable).

Either go to "C:\Users\USERNAME\AppData\Roaming\PopstarDevs\2Take1Menu\scripts"
or
Press Windows key + R and type "%appdata%/popstardevs/2take1menu/scripts" (without the quotes) and press enter.

Then just open the "Kek's menu 0.4.*.*.zip" and extract everything to your scripts folder.

(extra step for beta users, when in-game open the script and enable "participate in betas")
