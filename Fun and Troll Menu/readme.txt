Fun menu + Troll menu v1.0.1

Installtion guide: 

Choose a version you wanna use (latest version is recommended (v.0.5.1).

Either go to "C:\Users\USERNAME\AppData\Roaming\PopstarDevs\2Take1Menu\scripts"
or
Press Windows key + R and type "%appdata%/popstardevs/2take1menu/scripts" (without the quotes) and press enter.

Then just take everything from "Fun Menu v0.*.*" and drop it into your scripts folder.
